## About This Project

Nacho Website is a modern, personal portfolio website built to showcase my work, skills, and experience in AI, data science, and software engineering. The site is fully responsive, visually engaging, and designed for a seamless user experience.

---

## ✨ Features

- **Personal Branding:** Custom hero section, About, and Contact with my story and links.
- **Projects Gallery:** Interactive cards for real-world AI/ML projects, each with live demos and GitHub links.
- **Blog & Insights:** Space for sharing thoughts on AI, technology, and innovation.
- **Experience Timeline:** Visual timeline of education, certifications, and work experience.
- **Skills & Tools:** Animated skill badges and icons for technologies I use.
- **Modern UI:** Built with shadcn-ui, Tailwind CSS, and Framer Motion for smooth animations.
- **Dark Mode:** Fully supported for night owls and accessibility.
- **Custom Favicon:** Unique icon for a personal touch.

## 📦 Getting Started

```bash
# Clone the repository
$ git clone <https://github.com/HiNacho/Nacho_ai_lab.git>
$ cd <Nacho_ai_lab>

# Install dependencies
$ npm install

# Start the development server
$ npm run dev
```

---

## 📚 Project Structure

- `src/components/` — All UI components (Hero, About, Projects, Blog, etc.)
- `src/assets/` — Images and icons
- `src/pages/` — Main page and routing
- `public/` — Static files (favicon, resume, etc.)
- `tailwind.config.ts` — Tailwind CSS configuration
- `vite.config.ts` — Vite build configuration

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
